Regression Analysis

Why?

I created this page because I was looking for online tools that have full-scale regression analysis.
They were rare, and most of them didn't have all the statistics I needed. Also, I thought it would be good practice for me to make this applet.

How?

Insert your data in the corresponding boxes. The data can be seperated by commas, spaces, or a mix of them. Even multiples of them will work.
You can change the NAME of each variable in the LEFT BOX, and the DATA in the RIGHT BOX.
There is a limit to the number of independent variables you can add, which is 10.
Please, make sure that the number of observation is larger than the number of independent variables.
Also, make sure that the number of observation is exactly the SAME for all variables.
Finally, the independent variables MUST be independent (linear algebra independent).
Press "Regression" to see the regression analysis. It will only work when the data is valid.
Also, press "Plot" to plot the points and the fit line (each independent vs. response).
You'll have to press "Regression" whenever you change your data and want to plot again.
